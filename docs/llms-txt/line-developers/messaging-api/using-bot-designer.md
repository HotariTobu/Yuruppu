# LINE Bot Designer

<!-- note start -->

**Development of the LINE Bot Designer has been terminated**

As announced on [January 18, 2022](https://developers.line.biz/en/news/2022/01/18/development-of-line-bot-designer-has-been-finished/), LINE Bot Designer is no longer maintained. We recommend that you use the [Flex Message Simulator](https://developers.line.biz/flex-simulator/) to create [Flex Message](https://developers.line.biz/en/reference/messaging-api/#flex-message) prototypes.

<!-- note end -->

LINE Bot Designer is a tool for creating LINE Bot prototypes to design your own chatbots easily and quickly. You're not required to have any programming background use this tool.

![LINE Bot Designer](https://developers.line.biz/media/messaging-api/using-bot-designer/bot-designer.png)

## Download LINE Bot Designer 

LINE Bot Designer is available for free of charge. You only need to log in with your LINE account when you launch the tool. For the user guide, see this [manual](https://vos.line-scdn.net/line-developers/docs/media/messaging-api/using-bot-designer/bot-designer-user-manual.pdf) (PDF).

Download LINE Bot Designer from the [download page](https://developers.line.biz/en/docs/messaging-api/download-bot-designer/).

## Recommended environment 

We recommend these environment versions to use LINE Bot Designer:

- Windows 10
- macOS (Sierra or later)
